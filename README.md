Smart File Activity Anomaly Detection via IP Analytics with Decoy Data
Overview

Smart File Activity Anomaly Detection via IP Analytics with Decoy Data is a cloud-based cybersecurity solution designed to protect sensitive files from unauthorized access, insider threats, and data theft. The system continuously monitors user activities using IP analytics and behavioral profiling to identify suspicious access patterns. When abnormal behavior is detected, it automatically redirects attackers to encrypted decoy files while protecting the original data.

The project combines intelligent authentication, IP monitoring, anomaly detection, encryption, and deception technology to provide a secure and scalable cloud file protection system.

Problem Statement

Traditional cloud storage systems primarily rely on passwords and basic authentication mechanisms, making them vulnerable to insider attacks, credential theft, and unauthorized access. Existing security solutions often fail to detect abnormal user behavior in real time.

This project addresses these challenges by analyzing file access patterns, identifying suspicious activities, and protecting sensitive information using encrypted decoy data.
